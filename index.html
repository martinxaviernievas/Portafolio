Esta aplicación realiza una Transformada Rápida de Fourier en los datos proporcionados por la API de la Red Eléctrica de España (REE). Para ello, utiliza el módulo numpy.fft de la librería NumPy. La aplicación permite al usuario ingresar un rango de fechas y obtiene los datos de la API correspondientes a ese rango. Luego, aplica la Transformada Rápida de Fourier a los datos y devuelve un JSON con los datos obtenidos.
